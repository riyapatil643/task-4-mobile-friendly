# Task 4 - Mobile Friendly Website (Media Queries)

Converted a desktop-only webpage into a mobile-friendly responsive layout using CSS media queries.

## Changes Made
- Added media query for max-width: 768px
- Stacked columns vertically on mobile
- Reduced font sizes for readability
- Made navigation stack vertically
- Fixed overflow / horizontal scrolling
- Made images responsive using max-width: 100%

## Tools Used
- VS Code
- Chrome DevTools
